# Építésszervezés oktatói platform – V3 start

GitHub Pages-re feltölthető statikus oktatói platform magasépítő technikus építésszervezés tananyaghoz.

## Mi változott a V2-höz képest?

A V2 főként működő platformváz volt. A V3 start már megkezdi a tényleges tananyagbővítést:

- bővített tanulói olvasmány az első modulokhoz,
- részletesebb magyarázó szakaszok,
- kulcsgondolat blokk,
- fogalomcsoportok,
- tanulói feladatok,
- tanári használati javaslat,
- bővített kérdésbank.

## Bővített modulok ebben a csomagban

- 00 – Bevezető és fogalomtár
- 01 – A beruházási folyamat szereplői és feladataik
- 02 – Beruházások szervezése és előkészítése

A további modulok a V2 szerkezetével továbbra is elérhetők, később ugyanezzel a mintával bővíthetők.

## Feltöltés GitHubra

A meglévő repository-ban cseréld/frissítsd ezeket:

- `index.html`
- `assets/css/style.css`
- `assets/js/app.js`
- `data/platform-data.js`
- `data/module-content-v3.js`
- `assets/img/` mappa
- `docs/` mappa

A `data/module-content-v3.js` új fájl, ezt mindenképpen töltsd fel.
