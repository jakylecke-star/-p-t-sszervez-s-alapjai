# Építésszervezés oktatói platform

Statikus GitHub Pages kompatibilis oktatói platform magasépítő technikus építésszervezés tananyaghoz.

## Fájlstruktúra

- `index.html` – a platform belépőoldala és egyoldalas tanulói felülete
- `assets/css/style.css` – teljes arculat és reszponzív elrendezés
- `assets/js/app.js` – modulmegjelenítés, keresés, gyakorló kalkulátorok
- `data/platform-data.js` – modulok, fogalomtár, kérdések és gyakorlók adatai
- `assets/img/` – javított, olvasható infografikák
- `docs/github_pages_lepesek.md` – rövid feltöltési útmutató

## Használat GitHubon

1. Hozz létre egy új GitHub repository-t.
2. Töltsd fel a teljes mappatartalmat a repository gyökerébe.
3. Kapcsold be a GitHub Pages publikálást a repository beállításainál.
4. A `main` branch `/root` mappáját add meg forrásnak.

## Javasolt további fejlesztések

- fejezetenként teljes tanulói oldal
- Redmenta/Quizizz export kérdésbank
- letölthető jegyzőkönyv- és költségvetéssablonok
- tanári nézet külön óravázlattal
