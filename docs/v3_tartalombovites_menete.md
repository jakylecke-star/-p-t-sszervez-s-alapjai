# V3 tartalombővítés menete

## 1. Modulonkénti bővítés
Minden modulhoz ugyanaz a szerkezet készüljön:

1. áttekintés,
2. kulcsgondolat,
3. részletes tananyagszakaszok,
4. fogalomcsoportok,
5. tanulói feladatok,
6. tanári használati javaslat,
7. bővített kérdésbank.

## 2. Hol van az új tartalom?
Az új részletes tananyag a következő fájlban található:

`data/module-content-v3.js`

Ez külön van választva a V2 alapadatoktól, így a platform biztonságosan bővíthető anélkül, hogy a teljes alapadat-fájlt át kellene írni.

## 3. Következő bővítési kör
A következő modulok javasolt sorrendben:

- 03 – Mennyiségszámítás
- 04 – Költségvetés készítés
- 05 – Építési organizáció
- 06 – Időtervezés
