# GitHub Pages feltöltési lépések

1. Nyiss egy új repository-t, például: `epitesszervezes-platform`.
2. A letöltött ZIP-et csomagold ki.
3. A kicsomagolt mappa tartalmát töltsd fel a repository gyökerébe.
4. A GitHubon menj a repository `Settings` → `Pages` részéhez.
5. Válaszd a `Deploy from a branch` lehetőséget.
6. Branch: `main`, folder: `/root`.
7. Mentés után pár perc múlva megjelenik a publikus oldal linkje.

Megjegyzés: a forrás-PDF-et csak akkor tedd nyilvános repository-ba, ha a nyilvános megosztásra is van jogosultságod.
